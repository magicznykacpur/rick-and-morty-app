# Rick and Morty App

React + TS app that uses `https://rickandmortyapi.com` api to fetch and display data.

## Techologies used

### Styling

- `tailwindcss`
- `shadcn`

### Tanstack

- `react-query`
- `react-table`
- `react-router`

## How to run

- `npm i`
- `npm run dev` - app runs on port `42069`
